# 工具链

## vite

`npm create vite@latest`或`yarn create vite`或`pnpm create vite`
`pnpm i`

项目结构

```bash
react-vite-test
├── public
│   └── vite.svg
├── src
│   ├── assets
│   │   └── react.svg
│   ├── App.css
│   ├── App.tsx
│   ├── index.css
│   ├── main.tsx
│   └── vite-env.d.ts
├── index.html
├── package.json
├── pnpm-lock.yaml
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```

初始化 App.jsx

```jsx
function App() {
  return <></>
}
export default App
```

删除 App.css
删除 index.css
删除 main.jsx 中的 `import './index.css'`

安装 less `pnpm i less -D` 或 sass `pnpm i sass -D`

`npx start`
