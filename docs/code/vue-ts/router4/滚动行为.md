# 滚动行为

```ts
const router = createRouter({
  history: createWebHistory(),
  scrollBehavior: (to, from, savePosition) => {
    if (savePosition) {
      return savePosition
    } else {
      return {
        top: 0, // router4 中使用top和left定位
      }
    }
  },
  routes,
})
```

或者

```js
const router = createRouter({
  history: createWebHistory(),
  scrollBehavior: (to, from, savePosition) => {
    return {
      top: 500,
    }
  },
  routes,
})
```

或者

```js
const router = createRouter({
  history: createWebHistory(),
  scrollBehavior: (to, from, savePosition) => {
    return new Promise(r => {
      setTimeout(() => {
        r({
          top: 999999,
        })
      }, 2000)
    })
  },
  routes,
})
```
