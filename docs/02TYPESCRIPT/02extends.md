## extends

extends 关键字在不同场景下代表的含义不一样

1. 表示继承/拓展的含义
   js 中，和 class 一起使用，继承父类的方法和属性
   ts 中，还可以继承/拓展类型和接口
2. 表示约束的含义
   书写泛型的时候，作为对类型参数的约束，`<T extends B>`表示 T 被约束为必须是对 B 的继承/拓展
3. 表示分配的含义
   `<A extends B>`指类型 A 可以分配给类型 B，B 有的类型约束，A 都有

特殊：never 被认为是空的联合类型，是所有类型的子类型
